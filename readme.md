# DwarfsGui

Dwarfs 文件系统 Windows 图形化管理工具，提供便捷的镜像制作、挂载和提取功能。

## 简介

Dwarfs 是一个高压缩比的只读文件系统，专为游戏和大型数据集打包设计。DwarfsGui 为 Windows 用户提供了直观的图形界面，无需命令行操作即可完成：

- **制作镜像**：将文件夹或文件打包成高压缩比的 `.dwarfs` 镜像文件
- **挂载镜像**：将 `.dwarfs` 镜像挂载为虚拟驱动器，像普通文件夹一样访问
- **提取镜像**：从 `.dwarfs` 镜像中提取全部内容到指定目录

## 典型用途

### 游戏资源打包
游戏开发者和玩家可以将庞大的游戏资源文件夹打包成单个 `.dwarfs` 文件，大幅减少存储空间占用。挂载后即可正常访问，无需解压。

### 数据集分发
机器学习、科研等领域的大型数据集可通过 Dwarfs 格式分发，既保持高压缩率，又支持直接挂载使用，避免漫长的解压等待。

### 软件便携化
将软件及其依赖打包成镜像文件，挂载即可运行，适合便携式应用场景。

## 系统要求

- Windows 10/11
- [.NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)
- [WinFsp](https://winfsp.dev/)（挂载功能必需）

## 安装

1. 安装 WinFsp（挂载镜像必需）
2. 将 `DwarfsBin` 文件夹（包含 `mkdwarfs.exe`、`dwarfs.exe`、`dwarfsextract.exe`）放置在程序目录下
3. 运行 `DwarfsGui.exe`

## 使用方法

### 界面操作

程序分为三个功能标签页：

#### 制作镜像
1. 选择输入路径（文件或文件夹）
2. 设置输出镜像文件路径（`.dwarfs` 格式）
3. 配置压缩参数（压缩级别、是否去重等）
4. 点击「开始制作」

#### 挂载镜像
1. 选择 `.dwarfs` 镜像文件
2. 设置挂载点（留空则自动分配）
3. 配置缓存大小等参数
4. 点击「开始挂载」
5. 挂载完成后可在资源管理器中访问
6. 点击「卸载全部」可卸载所有已挂载镜像

#### 提取镜像
1. 选择 `.dwarfs` 镜像文件
2. 设置提取目标目录
3. 点击「开始提取」

### 右键菜单集成

点击「注册右键菜单」后，可在资源管理器右键菜单中直接调用：

- **制作镜像**：右键任意文件或文件夹，选择「制作镜像」
- **挂载镜像**：右键 `.dwarfs` 文件，选择「挂载镜像」

注册后可能需要重启资源管理器或重新登录才能生效。

### 命令行调用

程序支持命令行参数，方便右键菜单或脚本调用：

```bash
# 制作镜像
DwarfsGui.exe --create "输入路径"

# 挂载镜像
DwarfsGui.exe --mount "镜像路径"

# 提取镜像
DwarfsGui.exe --extract "镜像路径"
```

## 配置说明

程序首次运行会自动创建配置文件 `dwarfs_settings.json`，保存以下设置：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| WinFspPath | WinFsp 安装路径 | `C:\Program Files (x86)\WinFsp\bin` |
| CompressionLevel | 压缩级别（0-7） | 2 |
| DisableDedup | 禁用去重（加快制作速度） | false |
| CacheSize | 挂载缓存大小 | 512M |
| AutoMountPoint | 自动分配挂载点 | true |

## 注意事项

- 关闭程序时会提示卸载所有已挂载镜像
- 如果程序异常退出，残留的挂载进程可通过「卸载全部」清理
- 高压缩级别会显著增加制作时间，建议日常使用级别 2-5
